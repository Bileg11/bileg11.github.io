4. Дээрх "Telegram Chat ID" талбарт оруулна
2️⃣ GitHub Token авах (AI Chat)
1. github.com → нэвтэрнэ
2. Профайл → Settings
3. Зүүн доод → Developer settings
4. Personal access tokens → Tokens (classic)
5. "Generate new token (classic)" → Ямар ч scope сонгохгүй → Generate
6. ghp_xxx... гэсэн token-г хуулна
3️⃣ Telegram Bot Token авах (өөрийн bot)
1. Telegram дээр @BotFather нээнэ
2. /newbot илгээнэ
3. Bot-ын нэр оруулна (жишээ: MyJarvisBot)
4. Username оруулна (жишээ: myjarvis_bot)
5. 123456:ABC-DEF... гэсэн token ирнэ — хуулна
* Хоосон үлдээвэл системийн bot ашиглана
